Five plugins, deliberately small

Capability primitives + a lifecycle that orchestrates them.

Three capability primitives (codex, agy, contexthub) you can call directly. A token-efficient triage that grades every task before delegation. And forge — a seven-phase lifecycle that routes each phase to the right specialist.

One session, the right agent

Stay in your Claude Code session and hand a task to whichever external agent is strongest for it — no copying context around by hand.

Available
codex

Delegate image generation, hard-reasoning offload, browser checks, and raw Codex CLI execution when Codex is the stronger tool for the task.

/codex:imagegen /codex:reason /codex:review /codex:playwright /codex:resume /codex:exec
Available
agy

Reach for Google's Antigravity CLI for what Claude Code cannot do: 1M-token long-context analysis on Gemini 3 Pro, image generation via Nano Banana Pro, and video generation via Veo.

/agy:longctx /agy:nanobanana /agy:veo /agy:exec
Available
contexthub

Stage a four-phase debate among Claude Code, Codex, and Antigravity on one hard prompt — independent answers, mutual critique, refinement, then synthesis. Converges on a final response that surfaces consensus and remaining disagreements.

/contexthub:converge
Available
triage

Grade every task low / medium / hard before delegation, and write a routing plan that downstream specialists read to set their model effort. The token-efficiency primitive — spend big effort only on hard work, not by default on everything.

/triage:grade
Available
forge

The multi-agent software lifecycle: spec, plan, tdd, review, verify, debug, ship. Seven phases, each routing to the strongest specialist (Claude, Codex, Antigravity, contexthub) and graded for difficulty by triage. Every phase writes an audit artifact to disk.

/forge:spec /forge:plan /forge:tdd /forge:review /forge:verify /forge:debug /forge:ship

Capabilities

What each plugin covers.

Three capability primitives are thin layers over real CLIs. Triage is the token-efficiency primitive. Forge is the lifecycle that calls every other plugin at the right phase.

codex

OpenAI Codex CLI.

  • Hard reasoning at xhigh (gpt-5.5)
  • Code review & refactoring at high
  • Headless browser (Playwright)
  • Image generation (gpt-image-2)
  • Session resume
  • Raw exec — default effort medium

agy

Google Antigravity CLI.

  • 1M-token long-context analysis (Gemini 3 Pro)
  • Image generation (Nano Banana Pro)
  • Video generation (Veo)
  • Raw exec passthrough

contexthub

Multi-agent debate.

  • 4-phase Delphi protocol
  • Claude + Codex + Antigravity
  • Independent answers, mutual critique, refinement
  • Synthesizes consensus + remaining disagreements

triage

Token-efficient task grading.

  • Decomposes a task into single or subtasks
  • Grades each low / medium / hard
  • Suggests specialist + model effort per task
  • Writes a JSON routing plan to disk

forge

The multi-agent software lifecycle.

  • spec — refine ideas, debate when contested
  • plan — Claude drafts, Codex stress-tests, agy verifies coverage
  • tdd — RED/GREEN/REFACTOR; Codex when stuck
  • review — three-way: Claude + Codex + agy
  • verify — tests + Playwright + blast radius
  • debug — triangulate root cause across three agents
  • ship — commit, retro, push with consent

Install

Six commands in Claude Code.

1. Add the marketplace 
/plugin marketplace add https://github.com/infroid/carefully-crafted-plugins
2. Install codex 
/plugin install codex@carefully-crafted-plugins
3. Install agy 
/plugin install agy@carefully-crafted-plugins
4. Install contexthub 
/plugin install contexthub@carefully-crafted-plugins
5. Install triage 
/plugin install triage@carefully-crafted-plugins
6. Install forge 
/plugin install forge@carefully-crafted-plugins

About codex setup The codex bridge auto-scaffolds starter Markdown under docs/carefully-crafted-plugins/{constraints,output-formats}/ the first time you use one of its skills. Run /codex:setup to re-scaffold or refresh the starter files later.
